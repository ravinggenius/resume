Web Standards

Whenever possible I validate against the strictest (published) XHTML. I structure markup with div tags (with semantic identifiers) and leave presentation in CSS. If necessary I will use conditional comments to target Internet Explorer with style overrides.

Freedom | Open Source

I am a firm believer in using free software whenever possible, and I encourage my friends to do the same. I have been happily using GNU/Linux at home for many years.

git | Subversion

I use git at home for personal or open-source projects and Subversion (via TortoiseSVN) at work. I much prefer git for the flexiblity it provides.
